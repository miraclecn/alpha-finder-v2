# Research Operating Model

## Research Unit

The unit of research is a sleeve candidate, not a factor.

Each sleeve candidate must be anchored to:

- one mandate
- one thesis family
- one descriptor set
- one executable horizon
- one portfolio role

## Promotion Ladder

1. Thesis intake
2. Descriptor underwriting
3. Residual target definition
4. Alpha construction
5. Cost and regime validation
6. Portfolio marginal test
7. Paper sleeve
8. Shadow live
9. Capitalized live
10. Decay watch or retirement

## Promotion Gate

A sleeve is promotable only if it improves the portfolio after:

- industry and style controls
- transaction cost assumptions
- turnover budget checks
- A-share tradeability filters
- correlation checks against existing sleeves

## First Release Research Focus

Prioritize the theses that match a personal data stack instead of the ones that merely look attractive on paper.

Production order:

- `trend_leadership` as the main alpha sleeve
- `fundamental_rerating` as a slow quality/value anchor
- `risk_regime_filter` as a portfolio overlay built from market breadth, benchmark trend, dispersion, and volatility

Deferred or quarantined:

- `earnings_underreaction`
- `flow_liquidity_reversal`
- `crowding_anti_consensus`
- any news or message-derived thesis

The reason is economic and operational, not aesthetic:

- medium-horizon price leadership is measurable with stable daily data
- slow fundamental anchoring is still usable with conservative lag rules
- fast event and crowding logic are too fragile under personal data constraints unless their timing and coverage are proven first

## Promotion Discipline Under Personal Data Constraints

For the first release, a sleeve is promotable only if it:

- is built mainly from `green` daily and structural datasets
- survives next-open execution and high-cost stress
- remains valid under universe perturbation and regime splits
- improves the total portfolio rather than only its standalone chart

Anything dependent on uncertain event timing stays outside production promotion, even if a backtest looks strong.
